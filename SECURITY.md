| Version | Supported          |
| ------- | ------------------ |
| V40.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability in **CYBERINT // OS**, please report it responsibly.

**DO NOT** create a public GitHub Issue for security bugs.

### How to report:
1. Email the developer directly (or DM on Telegram @GHOST_LUNG).
2. Describe the exploit or vulnerability.
3. Wait for a confirmation before publishing details.

We aim to acknowledge reports within 48 hours.

### Scope
* **In Scope:** XSS vulnerabilities, API Key leakage mechanisms, Proxy bypasses.
* **Out of Scope:** Issues related to the third-party API providers (Leakosint), Cloudflare infrastructure, or User error.
